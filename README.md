# CoachMVP v2

Updated static site for the CoachMVP offer.

## Included

- `index.html`
- `how-it-works.html`
- `styles.css`
- `assets/onlyev-example.webp`
- `assets/smef-example.webp`

## What changed

- Added a dedicated **How it works** page.
- Added a short process preview to the homepage.
- Added a cleaner FAQ section.
- Added example screenshots and live example links.
- Kept the site static and GitHub Pages friendly.

## Notes

- GitHub Pages is enough for v1.
- Vercel is optional.
- The homepage form still uses `mailto:` to open the user's email app.
- The copy keeps the responsibility line clear: you set the person up, but they handle leads, follow-up, compliance, and closing.
- I worded the research step around **transactional intent** and **domain direction** instead of pushing exact trademark-match domains.

## Suggested upload flow

1. Replace your current `index.html` with this one.
2. Upload `how-it-works.html`.
3. Replace `styles.css`.
4. Upload the `assets` folder.
5. Push to GitHub Pages.

## Suggested path

If your repo is already publishing `/coach-mvp/`, the new page will live at:

- `/coach-mvp/how-it-works.html`
