# CoachMVP v1

Simple, mobile-first landing page for testing the coaching offer.

## Files

- `index.html`
- `styles.css`

## Recommended offer test

- Start at **$1,000** as a founding rate.
- Keep it fixed-price.
- Raise to **$1,500-$2,000** after your first few paid calls or wins.
- Do **not** tie your fee to how much money they make.
- If you want a light follow-up option later, add a fixed add-on like one review call.

## Recommended positioning

Use **CoachMVP** as the brand for v1.

Use this line as the offer:

> Launch a one-car lead-gen site with me.

That keeps the brand flexible while making the outcome clear.

## GitHub Pages

This site works great on GitHub Pages.

1. Create a repo.
2. Upload `index.html` and `styles.css` to the repo root.
3. Go to **Settings → Pages**.
4. Set source to **Deploy from branch**.
5. Choose `main` and `/root`.
6. Save.

## Vercel

You do **not** need Vercel for v1.

If you want to use it anyway:

1. Import the repo into Vercel.
2. Use the default settings.
3. No build command is required.
4. No framework preset is required for this static version.

## Notes

- The form opens the visitor's email app with a prefilled message to `contact@onlyusedtesla.com`.
- If you later want better form handling, switch to a real form backend.
- Keep the **no guarantees** language on the page.
